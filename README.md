Customer Behavior Analysis: End-to-End Data Pipeline
Python (EDA) | MySQL | Power BI | Capstone Project
📌 Project Overview
This project provides a comprehensive analysis of customer behavior to identify key purchasing patterns, segment high-value users, and provide actionable insights for business growth. By leveraging an end-to-end data pipeline—from raw data cleaning to interactive visualization—this project demonstrates how data-driven decisions can optimize marketing strategies and improve customer retention.

🛠️ Tech Stack & Tools
Data Cleaning & EDA: Python (Pandas, NumPy, Matplotlib, Seaborn)

Database Management: MySQL (Relational Schema Design, Complex Joins, CTEs)

Data Visualization: Power BI (DAX, Data Modeling, Interactive Dashboards)

Environment: Jupyter Notebook / VS Code

🚀 Project Workflow
Phase 1: Exploratory Data Analysis (Python)
Performed deep-dive EDA to understand data distributions and ensure data integrity.

Data Cleaning: Handled missing values, removed duplicates, and corrected data types.

Statistical Analysis: Identified outliers in purchase frequency and order values.

Feature Engineering: Created new metrics such as Customer Tenure and Average Order Value (AOV).

Phase 2: Database Management & SQL Analysis
After cleaning, the data was structured and migrated into a MySQL relational database for scalable querying.

Schema Design: Developed a normalized database schema to ensure data consistency.

Advanced Querying: Used Common Table Expressions (CTEs) and Window Functions to calculate:

Customer Lifetime Value (CLV).

Month-over-Month (MoM) growth rates.

Churn rate identification based on last purchase dates.

Phase 3: Data Visualization (Power BI)
Developed a multi-page interactive dashboard to translate complex SQL results into business-ready visuals.

KPI Tracking: Real-time tracking of Total Revenue, Active Users, and Retention Rates.

Customer Segmentation: Visualized behavioral clusters (e.g., VIP vs. At-Risk customers).

Trend Analysis: Interactive time-series charts to identify seasonal peaks in customer activity.

📊 Key Insights & Business Impact
Segment Identification: Identified that the top 20% of customers contribute to 75% of total revenue.

Churn Prevention: Discovered a 15% drop-off in customer activity after the third month, suggesting the need for a targeted loyalty program.

Optimized Marketing: Pinpointed specific time windows (e.g., weekends between 6 PM - 9 PM) where customer engagement is at its highest.
